Page d'Exemple pour l'Intégration de la Chatbox

Ceci est une page web standard qui sert de conteneur pour notre composant de chatbox. La chatbox elle-même est chargée dans une iframe en bas à droite de l'écran.

La configuration (couleur, titre, etc.) est passée directement à l'iframe via des paramètres d'URL. C'est une méthode d'intégration robuste qui garantit que la chatbox est bien isolée du reste de la page.

Essayez de modifier les paramètres dans le src de l'iframe ci-dessous, par exemple : ?primaryColor=ff5733&position=bottom-left
